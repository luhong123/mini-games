# 🎮 小游戏合集

13 个幼儿认知游戏 + 15 个经典小游戏，全部为单 HTML 文件，零外部依赖，开箱即玩。

## 快速开始

直接用浏览器打开 `index.html`，或部署到任意静态服务器。

```bash
git clone https://github.com/luhong123/mini-games.git

# Python 服务器
python -m http.server 8080

# Node.js 服务器
npx serve .
```

### 幼儿游戏特点

- **童声语音**：语速 0.5（慢速清晰）+ 音调 1.6（童声）
- **真实照片**：认识事物/动物声音使用本地真实照片，认识颜色等使用实物图片
- **触觉反馈**：答对振动 50ms，答错振动 100ms
- **答对掌声**：Web Audio API 生成鼓掌音效
- **零挫败感**：答错温柔鼓励"再试试~"，无 Game Over
- **家长设置**：长按标题可调语速和显示数量
- **彩色粒子庆祝**：答对时彩色方块喷射动画
- **5 秒自动提示**：探索模式自动高亮并语音引导
- **iOS 兼容**：touchstart 解锁语音引擎，解决 Safari 限制

## 技术特点

- 纯 HTML / CSS / JavaScript，**零外部依赖**
- 深色护眼主题（`#1e1e2a`）
- 响应式布局（`clamp()`），适配手机、平板和桌面
- 触屏 + 鼠标双重支持
- 刘海屏安全区域适配（`safe-area-inset`）
- `localStorage` 保存最高分（经典游戏）
- `requestAnimationFrame` 动画循环（实时游戏）
- Web Speech API 中文童声语音
- Web Audio API 鼓掌 / 答错提示音
- `navigator.vibrate` 触觉反馈
- 图片：76 张本地真实照片 + 150+ 个 SVG 矢量图标
- 仓库仅 32MB，轻量易部署
- 覆盖认知、实物、语言、社交四大领域
