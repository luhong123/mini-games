# 🎮 小游戏合集

17 个经典小游戏 + 2 个幼儿认知游戏，全部为单 HTML 文件，零外部依赖，开箱即玩。

## 快速开始

直接用浏览器打开 `index.html`，或部署到任意静态服务器。

## 幼儿认知专区

| 游戏 | 说明 |
|------|------|
| 🌈 **认识颜色** | 点击色块听颜色名，支持自由探索和找颜色两种模式 |
| ⬛ **认识形状** | Canvas 绘制笑脸形状，语音引导认识圆形/三角形/正方形等 |

特点：Web Speech API 中文语音、彩色粒子庆祝动画、零挫败感设计

## 技术特点

- 纯 HTML / CSS / JavaScript，**零外部依赖**
- 深色主题，统一视觉风格
- 响应式布局，适配手机和桌面
- 触屏 + 鼠标双重支持
- 使用 `localStorage` 保存最高分
- `requestAnimationFrame` 动画循环（实时游戏）
- Web Speech API 中文语音（幼儿游戏）

## 部署

```bash
# 克隆到本地
git clone https://github.com/luhong123/mini-games.git
```

直接双击 `index.html` 即可游玩，或使用任意静态服务器：

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```
