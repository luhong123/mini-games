# 🎮 小游戏合集

20 个经典小游戏 + 10 个幼儿认知游戏，全部为单 HTML 文件，零外部依赖，开箱即玩。

## 快速开始

直接用浏览器打开 `index.html`，或部署到任意静态服务器。

## 🧒 幼儿认知专区

专为 2-7 岁儿童设计，全部使用童声语音 + SVG 卡通图片。

| 游戏 | 说明 |
|------|------|
| 🌈 **认识颜色** | 7 种鲜艳颜色+实物图片，自由探索和找颜色模式 |
| ⬛ **认识形状** | 12 种形状 Canvas 绘制（含长方形/梯形/五边形等），都带笑脸 |
| 🐾 **认识事物** | 6 大品类 76 种：动物/蔬菜/用品/水果/车车/职业 |
| 🧍 **认识身体** | 16 种身体部位：眼睛鼻子嘴巴耳朵手等 |
| 🔢 **认识数字** | 1-10 数字+小动物数量对应计数 |
| 🔊 **动物声音** | 听"喵喵喵""汪汪汪"等叫声，找出对应的动物 |
| 🧣 **认识衣服** | 衣服裤子鞋子帽子书包等 10 种服饰 |
| 🍚 **认识食物** | 米饭面条面包牛奶鸡蛋等日常食物 |
| 📏 **大小长短** | 比较大小/长短/高矮，培养空间感知能力 |

### 幼儿游戏特点
- **童声语音**：语速 0.5（慢速清晰）+ 音调 1.6（童声）+ 优先中文女声
- **SVG 高清图片**：108 张本地卡通图片，白色圆底高对比度
- **触觉反馈**：答对振动 50ms，答错振动 100ms
- **答对掌声**：Web Audio API 生成鼓掌音效
- **零挫败感**：答错温柔鼓励"再试试~"，答错温和降调音效
- **家长设置**：长按标题可调语速和显示数量
- **彩色粒子庆祝**：答对时彩色方块喷射动画
- **5 秒自动提示**：探索模式自动高亮一个项目并语音引导

## 技术特点

- 纯 HTML / CSS / JavaScript，**零外部依赖**
- 深色主题（护眼色 `#1e1e2a`），统一视觉风格
- 响应式布局，适配手机、平板和桌面
- 触屏 + 鼠标双重支持
- 刘海屏适配（`safe-area-inset`）
- 使用 `localStorage` 保存最高分
- `requestAnimationFrame` 动画循环（实时游戏）
- Web Speech API 中文童声语音
- Web Audio API 鼓掌/提示音效
- `navigator.vibrate` 触觉反馈
- 所有图片为本地 SVG 矢量文件（共计 1.5MB）

## 部署

```bash
# 克隆到本地
git clone https://github.com/luhong123/mini-games.git
```

直接双击 `index.html` 即可游玩，或使用任意静态服务器：

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```
